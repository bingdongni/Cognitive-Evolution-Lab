# 认知进化实验室 - 示例数据说明

本目录包含认知进化实验室的示例数据文件，为各认知能力测试和演示提供标准化的数据支持。

## 📁 数据文件概览

### cognitive_test_data.yaml
**认知能力测试数据集合**
- **记忆测试数据**: 包含情景、语义、工作、程序记忆的标准化测试项目
- **推理测试数据**: 演绎、归纳、溯因、类比推理的案例集
- **创造力测试数据**: 发散思维、收敛思维、创意问题解决任务
- **观察力测试数据**: 视觉观察、模式识别、异常检测案例
- **注意力测试数据**: 选择性、持续性、分散性注意测试
- **想象力测试数据**: 情景想象、因果推理、时间想象任务
- **基准数据**: 人类和AI的表现基准，用于比较评估

### memory_data.yaml
**记忆系统详细示例数据**
- **情景记忆样本**: 包含情绪、感官细节的个人经历记录
- **语义记忆数据**: 概念关系、事实知识、层次结构表示
- **程序记忆内容**: 认知程序、运动技能、算法步骤
- **工作记忆缓存**: 当前处理信息、推理链、中间结果
- **记忆关联网络**: 不同记忆类型间的联想关系
- **遗忘曲线数据**: 记忆衰退时间模型和巩固效应

### reasoning_cases.yaml
**推理案例和测试数据集**
- **演绎推理案例**: 三段论、条件推理、选言推理标准案例
- **归纳推理案例**: 简单枚举、科学归纳、统计推理示例
- **溯因推理案例**: 最佳解释、诊断推理、因果推理案例
- **复合推理案例**: 连锁推理、并行推理、多步复杂推理
- **推理评估基准**: 人类表现标准、认知负荷评级、难度分析

### creativity_tasks.yaml
**创造力任务和评估标准**
- **发散思维任务**: 替代用途测试、后果测试、情境反应测试
- **收敛思维任务**: 远程联想测试、洞察问题、概念组合
- **创意问题解决**: 设计挑战、跨域创新、约束创新任务
- **协作创意任务**: 头脑风暴规则、团队创新过程
- **创意评估标准**: Torrance创造力评分、专家评判标准

## 🎯 数据特点

### 标准化格式
所有数据文件使用YAML格式，便于阅读和修改：
```yaml
# 示例：记忆测试数据格式
memory_test_data:
  episodic_memory:
    test_items:
      - content: "测试记忆内容"
        timestamp: 1703123456
        emotional_valence: 0.7
        personal_relevance: 0.8
```

### 分层结构
数据按认知能力和任务类型分层组织：
- **顶级分类**: 认知能力类型（记忆、推理、创造力等）
- **二级分类**: 具体任务或子模块
- **三级分类**: 测试项目或具体案例
- **属性字段**: 每项数据的详细参数

### 多维度标注
每个数据项包含多维度标注：
- **内容描述**: 文本、图像、概念等具体内容
- **元数据**: 时间戳、位置、上下文信息
- **评估指标**: 难度、重要性、置信度等
- **关系信息**: 与其他数据的关联关系

## 📊 数据使用指南

### 加载数据
```python
import yaml
from pathlib import Path

# 加载认知测试数据
with open('data/examples/cognitive_test_data.yaml', 'r', encoding='utf-8') as f:
    test_data = yaml.safe_load(f)

# 访问特定类型的测试
memory_tests = test_data['memory_test_data']
reasoning_cases = test_data['reasoning_test_data']
```

### 数据验证
```python
# 验证数据完整性
def validate_test_data(data):
    required_sections = ['memory_test_data', 'reasoning_test_data', 
                        'creativity_test_data', 'observation_test_data',
                        'attention_test_data', 'imagination_test_data']
    
    for section in required_sections:
        assert section in data, f"缺少必要的数据段: {section}"
    
    print("✅ 数据验证通过")
```

### 自定义扩展
```python
# 添加新的测试案例
def add_custom_test(data, test_type, new_case):
    if test_type not in data:
        data[test_type] = {}
    
    # 根据测试类型添加相应字段
    if test_type == 'memory_test_data':
        data[test_type]['custom_memory_tests'].append(new_case)
    
    # 保存更新后的数据
    with open('custom_test_data.yaml', 'w', encoding='utf-8') as f:
        yaml.dump(data, f, ensure_ascii=False, indent=2)
```

## 🧠 认知能力数据详解

### 记忆系统数据
**存储类型**: 多种记忆类型的示例
**关键字段**:
- `content`: 记忆内容（文本或概念）
- `type`: 记忆类型（episodic, semantic, procedural, working）
- `strength`: 记忆强度（0.0-1.0）
- `associations`: 相关联的记忆ID
- `accessibility`: 可访问性评分

**使用场景**:
- 记忆系统初始化和配置
- 记忆检索算法测试
- 遗忘曲线模型验证
- 记忆联想网络构建

### 推理系统数据
**推理类型**: 演绎、归纳、溯因、类比推理
**关键字段**:
- `premises`: 推理前提（列表）
- `conclusion`: 推理结论
- `validity`: 推理有效性
- `difficulty`: 难度评级
- `confidence`: 推理置信度

**使用场景**:
- 推理算法性能测试
- 认知负荷评估
- 推理准确性验证
- 推理类型比较研究

### 创造力系统数据
**任务类型**: 发散思维、收敛思维、创意解决
**关键字段**:
- `prompt`: 创意任务描述
- `expected_responses`: 期望响应数量
- `evaluation_criteria`: 评估标准
- `scoring_weights`: 评分权重
- `sample_responses`: 示例回答

**使用场景**:
- 创意生成算法测试
- 创造力评估模型验证
- 创意训练任务设计
- 跨领域创新研究

### 观察系统数据
**观察类型**: 视觉模式、异常检测、时间分析
**关键字段**:
- `observation_type`: 观察类型
- `target_elements`: 目标元素描述
- `distractors`: 干扰项信息
- `accuracy`: 准确率指标
- `reaction_time`: 反应时间

**使用场景**:
- 视觉识别算法测试
- 异常检测模型验证
- 注意力引导机制测试
- 观察力评估基准

### 注意力系统数据
**注意类型**: 选择性、持续性、分散性注意
**关键字段**:
- `task_description`: 注意任务描述
- `attention_weights`: 注意力权重分配
- `interference_level`: 干扰程度
- `duration`: 持续时间
- `performance_metrics`: 表现指标

**使用场景**:
- 注意力机制算法测试
- 注意控制能力评估
- 认知负荷测量
- 多任务处理验证

### 想象力系统数据
**想象类型**: 情景、因果、时间、创意想象
**关键字段**:
- `context`: 想象情境描述
- `constraints`: 想象约束条件
- `scenario_elements`: 场景元素
- `creativity_score`: 创意评分
- `plausibility`: 可信度评估

**使用场景**:
- 情景构建算法测试
- 创意想象能力评估
- 反事实推理验证
- 时间想象研究
