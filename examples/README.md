# 认知进化实验室 - 演示脚本与示例数据

本目录包含认知进化实验室的演示脚本、示例配置文件和测试数据，帮助用户快速上手并深入了解系统的各项功能。

## 📁 目录结构

```
examples/

└── evolution_experiment_config.yaml   # 进化实验配置

├── README.md                          # 数据目录说明

└── creativity_tasks.yaml             # 创造力任务数据集

├── demo.py                           # 主要演示脚本
└── quick_start.py                    # 快速入门脚本

```

## 🚀 快速开始

### 1. 运行快速入门演示
```bash
# 运行所有预设场景的快速入门演示
python scripts/quick_start.py

# 只运行基础认知测试
python scripts/quick_start.py --scenario basic

# 只运行创造力演示
python scripts/quick_start.py --scenario creative

# 运行进化演示
python scripts/quick_start.py --scenario evolution
```

### 2. 运行完整演示
```bash
# 运行六种认知能力的完整演示
python scripts/demo.py --mode full

# 运行特定认知能力测试
python scripts/demo.py --mode memory
python scripts/demo.py --mode reasoning
python scripts/demo.py --mode creativity
python scripts/demo.py --mode observation
python scripts/demo.py --mode attention
python scripts/demo.py --mode imagination

# 运行进化实验演示
python scripts/demo.py --mode evolution
```

### 3. 运行基础功能测试
```bash
# 运行所有基础功能测试
python tests/test_basic.py

# 运行详细测试
python tests/test_basic.py --verbose

# 只运行异步测试
python tests/test_basic.py --async_only
```

## 📋 认知能力演示内容

### 1. 记忆系统演示
- **情景记忆**: 测试个人经历和事件的存储与检索
- **语义记忆**: 测试事实知识和概念关系的记忆
- **工作记忆**: 测试短期信息处理和操作能力
- **程序记忆**: 测试技能和习惯的记忆
- **记忆联想**: 测试不同记忆类型之间的关联
- **遗忘曲线**: 测试记忆衰退和巩固机制

### 2. 推理能力演示
- **演绎推理**: 从一般到特殊的逻辑推理
- **归纳推理**: 从特殊到一般的事实归纳
- **溯因推理**: 最佳解释的推理过程
- **类比推理**: 结构映射和类比应用
- **条件推理**: 复杂条件语句的推理
- **多步推理**: 复合推理链的处理

### 3. 创造力演示
- **发散思维**: 创造性想法的生成能力
- **收敛思维**: 创意方案的评估和选择
- **创意问题解决**: 约束条件下的创新解决
- **新颖性评估**: 创新程度的量化评估
- **跨域创新**: 不同领域间的创意迁移
- **协作创意**: 群体创意生成过程

### 4. 观察力演示
- **视觉模式识别**: 图像中模式的识别能力
- **异常检测**: 偏离正常模式的异常识别
- **时间序列分析**: 随时间变化模式的分析
- **多尺度处理**: 不同空间尺度的信息处理
- **特征提取**: 关键视觉特征的提取
- **注意引导**: 基于目标的视觉注意

### 5. 注意力演示
- **选择性注意**: 在干扰中专注目标的能力
- **持续注意**: 长期维持注意的能力
- **分散注意**: 同时处理多个任务的能力
- **注意控制**: 注意焦点的转换和控制
- **注意分配**: 有限注意资源的分配策略
- **注意衰减**: 注意疲劳和恢复机制

### 6. 想象力演示
- **情景想象**: 未来场景的构造能力
- **因果想象**: 因果关系的推理和预测
- **时间想象**: 过去、现在、未来的时间想象
- **创意想象**: 突破常规的创新想象
- **反事实想象**: 替代情况的想象推理
- **多维想象**: 复杂三维情景的想象构建

### 7. 协同进化演示
- **单智能体进化**: 个体的遗传算法优化
- **多智能体进化**: 群体智能的协同进化
- **文化进化**: 知识和文化的社会传播
- **环境共演化**: 智能体与环境的相互作用
- **知识进化**: 概念和理论的发展演进
- **适应性学习**: 基于反馈的策略调整

## 🐛 常见问题

### Q: 演示运行时间过长怎么办？
A: 使用 `basic_cognition_config.yaml` 配置文件，或修改参数降低复杂度

### Q: 如何保存演示结果？
A: 所有演示结果会自动保存到指定目录，可在命令行指定 `--output` 参数

### Q: 可以在无GPU环境下运行吗？
A: 可以，系统会自动检测硬件并选择最佳配置

### Q: 如何添加新的测试任务？
A: 在对应的数据文件中添加新的测试项目，系统会自动识别
